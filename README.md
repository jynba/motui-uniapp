<h1 align="center">motui-uniapp</h1>
<p align="center">轻量级 uni-app 组件库，支持移动端 H5 和 小程序开发</p>

## 介绍

motui-uniapp 组件库，基于Taro版[`NutUi`](https://nutui.jd.com/#/) 4.x版本修改而来，适配了uni-app, 使用 Vue 技术栈开发小程序应用，开箱即用，拥有丰富的业务组件。

## 特性

- 🚀 高质量组件，覆盖移动端主流场景
- 💪 支持一套代码同时开发多端
- 🍭 支持按需引用
- 💪 支持 TypeScript
- 💪 支持动态定制主题
- 🍭 支持暗黑模式
- 🌍 支持国际化
